VISUAL STUDIO CODE

Visual Studio Code(Vs Code) is a lightweight butpowerful source code editor developed by Microsoft. It is a streamlined code editor with support for development operations like debugging, task running, and version control. It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflows to fuller featured IDEs, such as Visual Studio IDE.

Features of VScode:

  • Syntax Highlighting and Auto-Completion for most languages.
  • Integrated Terminalfor running scripts or commands.
  • Version Control Integrationfor Git and other systems
  • Extensions Marketplaceto add functionality, such as language specific support, debuggers, themes and more.
  • Customizable Interfacewith user-defined settings and shortcuts.
  • Built-in Debuggingfor a seamless coding experience.
